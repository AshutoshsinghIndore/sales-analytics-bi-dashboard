## Sales Analytics BI Dashboard

### Objective
Build an end-to-end sales analytics solution to analyze revenue, orders,
customers, and product performance using SQL and Power BI.

### Tech Stack
- SQL (MySQL)
- Power BI
- DAX
- Star Schema Data Modeling

### Data Modeling
- Fact Table: fact_orders
- Dimension Tables: dim_customers, dim_products, dim_employees, dim_date

### Key KPIs
- Total Revenue
- Total Orders
- Average Order Value
- Product-wise Sales
- Customer Contribution

### Dashboard Features
- Interactive slicers
- Drill-through analysis
- KPI cards & trend visuals

### Folder Structure
- /sql → transformations & views
- /pbix → Power BI dashboard
